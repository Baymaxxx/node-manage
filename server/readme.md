
`http://localhost:3000/api/`


关于静态资源不再提供补全 前端中自行补全 静态资源地址为: `http://statics.zhuishushenqi.com`


## api接口

#### 获取带书籍数量的父分类

url: `http://localhost:3000/api/categories`

#### 获取带子分类的分类

url: `http://localhost:3000/api/sub-categories`

#### 获取分类详情

url: `http://localhost:3000/api/category-info?gender=male&type=hot&major=奇幻&minor=&start=0&limit=20`

query:

```js
{
  gender, type, major(主分类), minor(子分类), start, limit
}
```

#### 获取书籍详情

url: `http://localhost:3000/api/book-info/53115e30173bfacb4904897e`

params:

```js
{
  id: book id
}
```

#### 获取书籍相关推荐

url: `http://localhost:3000/api/recommend/53115e30173bfacb4904897e`

query:

```js
{
  id: 书籍id
}
```

#### 获取作者名下的书籍

url: `http://localhost:3000/api/author-books?author=忘语`

query:

```js
{
  author: 作者名
}
```

#### 获取书籍源

url: `http://localhost:3000/api/book-sources?view=summary&book=567d2cb9ee0e56bc713cb2c0`

query:

```js
{
  view: summary,
  book: {bookid}
}
```

#### 获取书籍章节

url: `http://localhost:3000/api/book-chapters/56f8da09176d03ac1983f6cd`

params:

```js
{
  id: {书籍源id}
}
```

#### 获取章节详细内容

url: `http://localhost:3000/api/chapters/http%3A%2F%2Fvip.zhuishushenqi.com%2Fchapter%2F56f8da09176d03ac1983f6d7%3Fcv%3D1486473051386`  **注意这里需要进行url编码**

params:

```js
{
  link: {目录中的章节link}
}
```

#### 获取搜索结果

url: `http://localhost:3000/api/search?keyword=遮天`

query:

```js
{
  keyword: {书籍关键字}
}
```

#### 获取排名分类

url: `http://localhost:3000/api/rank-category`

#### 获取排名详情

url: `http://localhost:3000/api/rank/54d42d92321052167dfb75e3`

params:

```json
{
  id: {排名分类的id}
}
```
